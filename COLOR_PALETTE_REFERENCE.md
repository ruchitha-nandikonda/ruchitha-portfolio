## Usage Guidelines

### Primary Colors
- **Mauve Plum (#954F6C)**: Use for primary buttons, links, and brand elements
- **Dusty Rose (#DFB8B2)**: Use for hover states and secondary accents
- **Deep Muted Purple (#623B5B)**: Use for secondary text and subtle borders

### Text Colors
- **Primary Text**: Use for main content and headings
- **Secondary Text**: Use for supporting content and descriptions
- **Muted Text**: Use for captions, labels, and less important information

### Background Colors
- **Main Background**: Use for page backgrounds and large areas
- **Secondary Background**: Use for cards, modals, and elevated surfaces

### Dark Mode Considerations
- All colors automatically switch in dark mode
- Purple tones become more vibrant in dark mode
- Text colors invert for proper contrast
- Backgrounds use deep wine colors for sophistication

## Accessibility

### Contrast Ratios
- All text colors meet WCAG AA standards (4.5:1 minimum)
- Primary brand colors provide sufficient contrast on both light and dark backgrounds
- Status colors are chosen for accessibility and colorblind users

### Color Blindness
- The palette works well for users with various types of color blindness
- Status colors use both color and other visual cues (icons, text)
- Important information is never conveyed through color alone

## Implementation

### Using Colors in CSS
```css
/* Use semantic variables */
.my-element {
  color: var(--text-primary);
  background-color: var(--clr-background);
  border-color: var(--clr-primary);
}

/* Use specific color variables when needed */
.my-accent {
  color: var(--clr-primary-200);
  background-color: var(--clr-primary-50);
}
```

### Adding New Colors
1. Add the color to the appropriate section in `base.css`
2. Use semantic naming (e.g., `--clr-success`, `--clr-warning`)
3. Add dark mode variants if needed
4. Update this reference document
5. Test accessibility and contrast ratios

## Color Psychology

### Wine & Mauve Theme
- **Sophisticated**: The wine colors convey elegance and professionalism
- **Creative**: Purple tones suggest innovation and creativity
- **Trustworthy**: Deep, muted tones build confidence
- **Modern**: Contemporary color combinations feel current and fresh

### Dark Mode Benefits
- **Reduced Eye Strain**: Dark backgrounds are easier on the eyes
- **Battery Life**: Dark themes can save battery on OLED displays
- **Focus**: Dark backgrounds help content stand out
- **Modern Feel**: Dark themes are associated with cutting-edge design
