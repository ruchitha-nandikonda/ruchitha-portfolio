SmartCart

SmartCart is a full-stack grocery optimizer that keeps your pantry, meal plans, and local deals in sync so you only buy what you need.

  • React + TypeScript frontend with Tailwind for a responsive UI.
  • Spring Boot backend exposing REST APIs for pantry, deals, and lists.
  • AWS DynamoDB for storing pantry and pricing data.
  • AWS Textract for scanning grocery receipts and updating the pantry automatically.
  • Docker and Terraform for repeatable cloud deployment.

Lucid Loom

Lucid Loom is an AI-powered dream platform that turns user dreams into structured insights and visuals.

  • React + Tailwind UI built with Vite for fast, smooth interactions.
  • Python FastAPI backend with JWT auth, SQLAlchemy models, and WebSockets.
  • Integrations with Groq and DALL-E for analysis and image generation.
  • PostgreSQL in production, Docker-based deployment on Railway and Vercel.
  • Focus on async workflows, API design, and clear separation of concerns.
