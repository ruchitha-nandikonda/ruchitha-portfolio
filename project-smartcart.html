SmartCart

A full-stack grocery planning platform that syncs your pantry, meal plans, and local deals to generate cost-efficient shopping lists using AI-powered receipt scanning.

Overview

Grocery shopping often leads to overspending and food waste. People buy items they already have, miss deals, and struggle to plan meals efficiently. Without a system to track pantry inventory, compare prices, and optimize shopping lists, consumers waste money and time.

SmartCart solves this by automating pantry management through receipt scanning, syncing meal plans with inventory, and generating optimized shopping lists that factor in current deals and existing stock. The platform uses AWS Textract to extract line-item data from receipts, Spring Boot for intelligent optimization, and React for a modern, responsive interface.

The Challenge

Manual Pantry Tracking

Users struggle to manually track what's in their pantry, leading to duplicate purchases, expired items, and wasted food. There's no automated way to update inventory after shopping.

Inefficient Meal Planning

Meal planning doesn't account for existing pantry items, resulting in unnecessary purchases. Users can't easily see what ingredients they already have versus what they need to buy.

Missed Deals and Overspending

Without a system to compare deals across stores and factor in current inventory, users miss opportunities to save money and often buy items at higher prices than necessary.

Complex Optimization Logic

Generating an optimal shopping list requires complex calculations: subtracting pantry inventory, evaluating multiple store deals, considering meal dependencies, and prioritizing cost efficiency.

The Solution

I built a full-stack platform that automates pantry management, optimizes shopping lists, and integrates meal planning with inventory tracking. The solution combines AWS cloud services, Spring Boot optimization logic, and a modern React frontend.

Automated Receipt Processing

Users upload grocery receipts to S3, and AWS Textract OCR automatically extracts line-item data including product names, quantities, and prices. This data is processed and stored in DynamoDB, updating the pantry inventory without manual entry.

Real-Time Pantry System

Built on AWS DynamoDB, the pantry tracks:

  • Stocked Items: Current inventory with quantities and purchase dates.
  • Expiration Tracking: Alerts for items approaching expiration to reduce waste.
  • Meal Dependencies: Tracks which meals require which ingredients for intelligent planning.
  • Low Stock Alerts: Notifications when items are running low.

Intelligent Optimizer Engine

The Spring Boot backend includes a Java-based optimizer that:

  • Subtracts Pantry Inventory: Removes items already in stock from shopping requirements.
  • Evaluates Store Deals: Compares prices across multiple stores and selects the best deals.
  • Generates Cost-Efficient Lists: Creates optimized shopping lists prioritizing savings and necessity.
  • Meal Planning Integration: Calculates required ingredients based on selected recipes and existing pantry items.

Modern Dashboard Experience

The React frontend features glassmorphism design, gradient layers, animations, and visual pantry tiles. Built with Redux Toolkit for state management, custom hooks for reduced-motion detection, and persistent user preferences in localStorage.
